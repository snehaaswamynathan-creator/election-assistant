# Election Assistant 🗳️

AI assistant to help users understand election timelines, find polling booths, and get voting FAQs answered.

## Features
- **AI Chatbot**: Ask election-related questions in plain English
- **Interactive Timeline**: See key dates for registration, campaigning, voting day
- **Booth Finder**: Enter pincode → get nearest polling station with map
- **FAQ Section**: Instant answers on voter ID, documents needed, etc

## Tech Stack
**Frontend**: React, Vite, TailwindCSS  
**Backend**: Node.js, Express  
**APIs**: OpenAI GPT-4, Google Maps API  
**Data**: Election Commission of India public datasets
